# FarmaGO

Proyecto desarrollado para la universidad utilizando Flask y MongoDB.

## 📌 Descripción

FarmaGO es una plataforma enfocada en la gestión y visualización de productos farmacéuticos.  
El objetivo del proyecto es ofrecer una interfaz sencilla e intuitiva para administrar información de medicamentos y usuarios.

## 🚀 Tecnologías utilizadas

- Python
- Flask
- MongoDB
- HTML
- CSS
- JavaScript
